This project cleans and prepares NHANES (National Health and Nutrition Examination Survey) data for statistical analysis.  
All data cleaning steps are implemented in R using the `tidyverse` and `NHANES` packages.

---

## Files in This Repository

- `nhanes_cleaning.qmd`  
  Quarto document containing all data cleaning steps.

- `nhanes_clean.rds`  
  Saved cleaned dataset generated after running the script.

- `README.md`  
  Instructions for running the project locally.

---

## How to Run This Project Locally

Follow the steps below to run this project on your own computer.

### 1. Download or clone the repository

git clone https://github.com/Klara1205/NHANES-project.git# NHANES-project

### 2. install.packages(c("tidyverse", "NHANES", "shiny", "ggplot2", "modelr"))

### 3. Run QMD file

after reading nhanes_cleaning.qmd you will have a nhane_clean.rds for app running
