This project cleans and prepares NHANES (National Health and Nutrition Examination Survey) data for statistical analysis.  
All data cleaning steps are implemented in R using the `tidyverse` and `NHANES` packages.

---

## Files in This Repository

- `nhanes_cleaning.qmd`  
  Quarto document containing all data cleaning steps.

- `nhanes_clean.rds`  
  Saved cleaned dataset generated after running the script.

- `README.md`  
  Instructions for running the project locally.

---
